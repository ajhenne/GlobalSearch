# GlobalSearch (ESO Addon)

A global search interface for *The Elder Scrolls Online*. This offers a lightweight and unified search bar to quickly find almost any item (sets, recipes, crafting materials and more).

## Features

- **Item Sets**: search across all sets within the game.

The goal of this addon is not to add a variety of features. It's focus is a quick interface to find and bring up the tooltip for any possible item in the game -- no more digging around multiple menus, addons or websites. Naturally this integrates with any addons you may have that can provide utility from the tooltip: quickly find sales data (e.g. MM, TTC), what character knowledge you have for that set, how many of that resource you have.

### Planned Features
- **More search categories:** crafting materials, recipes & plans, furnishings, collectibles.
- Quick toggle between plans and crafted materials.
- Link to chat feature.

## How to use

Adds two commands:
- `/gs` -- toggle the search window.
- `/gss` -- toggle the settings window.

Also provides a keybind in the setting to quickly bring up the search window.


## Dependencies

- LibSets
- LibAddonMenu-2.0


## Installation

1. Manual:
    - Download the `.zip` file.
    - Extract the contents into your `Documents/Elder Scrolls Online/live/Addons/` folder.
    - Ensure dependencies are also installed.
