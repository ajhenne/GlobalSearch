# GlobalSearch (ESO Addon)

A global search interface for *The Elder Scrolls Online*. This offers a lightweight and unified search bar to quickly find almost any item across a number of categories.

## Features

Search across a variety of categories:
- Item Sets
- Learnables: all recipes, motifs and other learnable items within the game.
- Furniture
- Materials

The goal of this addon is not to add a variety of features. It's focus is a quick interface to find and bring up the tooltip for any possible item in the game. No more digging around multiple menus, addons or websites. Naturally this integrates with any addons you may have that can provide utility from the tooltip: quickly find sales data (e.g. MM, TTC), what character knowledge you have for that set, how many of that resource you have.


### Planned Features
- More search categories, such as POIs and locations.
- Quick toggle between plans and crafted materials.
- Link to chat feature.

## How to use

Adds the command `/gs` to toggle the search window.

Also provides a bindable keybind in the setting to quickly bring up the search window.


## Dependencies

- LibSets
- LibAddonMenu-2.0


## Installation

1. Manual:
    - Download the `.zip` file.
    - Extract the contents into your `Documents/Elder Scrolls Online/live/Addons/` folder.
    - Ensure dependencies are also installed.

**Installation note:** a `GlobalSearchData` folder is also included, this stores cache for the item data. If you are installing manually, make sure this folder is also moved to your Addons folder.
